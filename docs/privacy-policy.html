Politique de confidentialité

Dernière mise à jour : mars 2026

En résumé : PiscineTimer ne collecte, ne stocke et ne transmet aucune donnée personnelle à des serveurs externes. Toutes vos données restent sur votre appareil.

1. Présentation

PiscineTimer est une application de chronométrage de natation composée de deux parties :

  • Une application Galaxy Watch (Wear OS) qui enregistre vos séances de nage
  • Une application téléphone Android compagnon qui affiche les statistiques

2. Données collectées

PiscineTimer collecte uniquement les données nécessaires à son fonctionnement :

  • Données de séances : durée, nombre de longueurs, temps par longueur, fréquence de nage (cadence)
  • Données de capteurs (optionnel, mode debug) : données brutes de l'accéléromètre et du gyroscope pour améliorer la détection automatique des virages

3. Stockage des données

Toutes les données sont stockées localement sur vos appareils :

  • Sur la montre : base de données Room (stockage interne de l'app)
  • Sur le téléphone : base de données Room (stockage interne de l'app)
  • La synchronisation entre montre et téléphone s'effectue via Bluetooth, sans passer par Internet

4. Données non collectées

PiscineTimer ne collecte pas :

  • Votre nom, email ou toute identité personnelle
  • Votre localisation GPS
  • Vos contacts, photos ou fichiers
  • Données de santé transmises à des tiers
  • Données publicitaires ou de tracking

5. Partage des données

Aucune donnée n'est partagée avec des tiers. PiscineTimer ne contient aucun SDK publicitaire, analytique ou de tracking.

6. Autorisations requises

  • BODY_SENSORS : accès à l'accéléromètre et au gyroscope de la montre pour la détection des virages
  • VIBRATE : retour haptique lors des passages
  • WAKE_LOCK : maintien de l'écran allumé pendant la nage

Ces autorisations sont utilisées uniquement en local, sur l'appareil.

7. Suppression des données

Pour supprimer toutes vos données, désinstallez l'application. Toutes les données locales seront effacées automatiquement.
